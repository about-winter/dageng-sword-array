# Claude Code 功能介绍网站

一个简洁美观的单页网站，用于介绍 Anthropic 官方 AI 编程助手 **Claude Code** 的主要功能。

## 🌐 在线预览

将网站部署到 GitHub Pages 后即可访问。

## ✨ 功能特性

- **响应式设计** - 适配桌面端和移动端
- **现代化 UI** - 渐变背景、卡片布局、悬停动画
- **六大核心功能** - 文件操作、代码搜索、命令执行、智能 Agent、依赖管理、安全检查
- **常用命令展示** - 快速了解 Claude Code 的快捷命令

## 📁 文件结构

```
.
├── index.html          # 主页面
└── README.md           # 项目说明
```

## 🚀 部署到 GitHub Pages

1. 在 GitHub 创建新仓库
2. 推送代码到仓库
3. 进入 Settings → Pages
4. 选择 `main` 分支作为源
5. 等待部署完成，获取访问链接

## 💻 本地预览

直接使用浏览器打开 `index.html` 文件即可查看效果。

## 📝 技术栈

- HTML5
- CSS3 (Grid 布局、动画、响应式)
- 原生 JavaScript (无依赖)
